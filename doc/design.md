# 3. Calm Authority (Spokój i Prestiż)

1. https://coolors.co/0b3d2e-dbc1ac-fafaf5-0f1f15

**Charakter:** Spokój, Harmonia, Pieniądze.
Ciemna zieleń kojarzy się z tradycyjnymi gabinetami, bibliotekami i... pieniędzmi. Jest mniej opatrzona niż granat, a wzbudza bardzo duże zaufanie. Beż ociepla wizerunek.

- **Kolor Główny:** Butelkowa Zieleń (`#1E392A` lub `#0B3D2E`).
- **Kolor Akcentowy:** Piaskowy Beż / Złoty Beż (`#DBC1AC`).
- **Tło Sekcji:** Kremowa Biel (`#FAFAF5`).
- **Tekst:** Bardzo ciemna zieleń lub czarny (`#0F1F15`).

### Czcionka

Zieleń i beż wymagają typografii, która wygląda jak z dobrego magazynu lub klasycznej książki prawniczej.

- **Nagłówki:** **Cormorant Garamond**
- https://fonts.google.com/specimen/Cormorant+Garamond
    - *Dlaczego:* Piękna, klasyczna czcionka szeryfowa o wysokim kontraście (cienkie i grube linie). Wygląda bardzo drogo i ekskluzywnie.
- **Tekst główny:** **Open Sans**
- https://fonts.google.com/specimen/Open+Sans
    - *Dlaczego:* Prosta, przyjazna i otwarta. Stanowi idealne tło dla ozdobnych nagłówków, nie odwracając od nich uwagi.
